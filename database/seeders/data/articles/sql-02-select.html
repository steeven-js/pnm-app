SELECT — Lire des données

La commande SELECT

SELECT est la commande la plus utilisée en SQL. Elle permet de lire (extraire) des données d'une table. C'est la seule commande que vous utiliserez pour investiguer les anomalies PNM.

Syntaxe de base

SELECT colonnes FROM table;

Lire toutes les colonnes

L'étoile * signifie « toutes les colonnes » :

SELECT * FROM porta_operateur;

Résultat : toutes les colonnes (code, nom, is_active, is_actor, contact, email, fax, directory, comment) pour chaque opérateur.

Lire des colonnes spécifiques

Vous pouvez choisir quelles colonnes afficher :

SELECT code, nom FROM porta_operateur;

Résultat : seulement le code et le nom de chaque opérateur.

Limiter le nombre de résultats : LIMIT

Quand une table contient des milliers de lignes, on utilise LIMIT pour n'en afficher qu'une partie :

SELECT * FROM porta_portage LIMIT 10;

Affiche les 10 premiers portages.

Trier les résultats : ORDER BY

ORDER BY trie les résultats par une colonne :

SELECT * FROM porta_fichier ORDER BY date_creation DESC LIMIT 20;

Affiche les 20 fichiers les plus récents (DESC = du plus récent au plus ancien).

Renommer une colonne : AS

AS donne un alias (surnom) à une colonne dans les résultats :

SELECT code AS code_operateur, nom AS nom_operateur
FROM porta_operateur;

Compter les lignes : COUNT

COUNT(*) compte le nombre total de lignes :

SELECT COUNT(*) AS total_portages FROM porta_portage;

Résultat : un seul nombre indiquant le total de portages en base.

Valeurs uniques : DISTINCT

DISTINCT élimine les doublons :

SELECT DISTINCT operateur_id_actuel FROM porta_msisdn;

Liste les codes opérateurs qui ont au moins un numéro attribué.

Exercices :
  1. Affichez tous les codes ticket : SELECT * FROM porta_code_ticket
  2. Comptez le nombre de fichiers : SELECT COUNT(*) FROM porta_fichier
  3. Affichez les 5 portages les plus récents : SELECT * FROM porta_portage ORDER BY portage_date_creation DESC LIMIT 5
  4. Listez les répertoires de fichiers distincts : SELECT DISTINCT repertoire FROM porta_fichier
