Architecture réseau — Vue d'ensemble

L'architecture réseau de Digicel Antilles-Guyane repose sur une infrastructure multi-technologies (2G/3G/4G) couvrant la Guadeloupe, la Martinique et la Guyane. Comprendre cette architecture est essentiel pour appréhender les impacts de la portabilité sur le routage des communications.

Les trois couches du réseau

Le réseau mobile se décompose en trois couches principales :

  • RAN (Radio Access Network) : antennes et stations de base assurant la couverture radio
  • Core Network : cœur de réseau gérant l'intelligence des services
  • Transport : liaisons fibre optique et faisceaux hertziens interconnectant les sites

Core Network — Cœur de réseau

Le cœur de réseau concentre les équipements critiques pour le traitement des appels et des données :

Domaine Circuit (2G/3G) :

  • MSC (Mobile Switching Center) : commutateur d'appels voix, gère l'établissement et le routage des appels
  • HLR (Home Location Register) : base de données centrale des abonnés, stocke les profils, les droits et la localisation
  • VLR (Visitor Location Register) : base temporaire des abonnés présents dans une zone MSC

Domaine Paquet (4G/LTE) :

  • EPC (Evolved Packet Core) : cœur de réseau 4G composé du MME, S-GW et P-GW
  • HSS (Home Subscriber Server) : équivalent 4G du HLR, base de données des abonnés LTE
  • IMS (IP Multimedia Subsystem) : plateforme de services voix sur IP (VoLTE), messagerie enrichie

Comment un appel est acheminé

Lorsqu'un abonné Digicel appelle un numéro, le flux suit ces étapes :

  • Le terminal s'accroche au réseau via le RAN (station de base la plus proche)
  • La demande d'appel remonte vers le MSC (voix 2G/3G) ou l'IMS (VoLTE)
  • Le MSC interroge le HLR pour déterminer l'opérateur hébergeant le numéro appelé
  • Si le numéro est porté, le HLR consulte les tables de routage PNM (mises à jour par MOBI) pour rediriger vers le bon opérateur
  • L'appel est acheminé via les interconnexions vers l'opérateur destinataire
  • L'opérateur destinataire route l'appel jusqu'au terminal appelé

Comment les données transitent

Pour le trafic data (navigation, applications, streaming) :

  • Le terminal établit un bearer (tunnel de données) avec l'EPC via le RAN 4G
  • Le P-GW (Packet Gateway) connecte l'abonné à Internet
  • Le HSS fournit le profil de l'abonné (débit autorisé, quota data, APN)
  • Les paquets sont acheminés de manière transparente — la portabilité n'affecte pas le trafic data sortant

Transport et interconnexions

Les trois territoires (Guadeloupe, Martinique, Guyane) sont reliés par des liaisons sous-marines et des réseaux de transport terrestres. Les interconnexions avec les autres opérateurs (Orange, SFR/Only) s'effectuent via des points d'interconnexion (POI) dédiés, essentiels pour le routage des appels vers les numéros portés.
