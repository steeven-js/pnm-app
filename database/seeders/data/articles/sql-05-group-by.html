GROUP BY — Compter et résumer

Pourquoi regrouper ?

En investigation PNM, on a souvent besoin de résumés : « combien de portages par opérateur ? », « combien de fichiers par jour ? ». GROUP BY permet de regrouper les lignes par valeur et d'appliquer des fonctions d'agrégation.

Fonctions d'agrégation

FonctionRôleExemple
COUNT(*)Compter les lignesNombre de portages
COUNT(DISTINCT col)Compter les valeurs uniquesNombre d'opérateurs différents
SUM(col)SommeRarement utilisé en PNM
AVG(col)MoyenneDurée moyenne d'un portage
MIN(col)Valeur minimumDate du premier portage
MAX(col)Valeur maximumDate du dernier portage

Syntaxe GROUP BY

SELECT colonne_de_regroupement, FONCTION_AGREGATION(*)
FROM table
GROUP BY colonne_de_regroupement;

Exemples PNM concrets

Nombre de portages par état

SELECT p.portage_etat AS etat, COUNT(*) AS total
FROM porta_portage p
GROUP BY p.portage_etat
ORDER BY total DESC;

Donne une vue d'ensemble : combien de portages sont initial, enCours, diffuse, termine, refuse, annule.

Nombre de fichiers par jour

SELECT date_creation, COUNT(*) AS nb_fichiers
FROM porta_fichier
GROUP BY date_creation
ORDER BY date_creation DESC;

Portages par opérateur donneur

SELECT o.nom AS operateur, COUNT(*) AS portages_sortants
FROM porta_dossier d
JOIN porta_operateur o ON o.code = d.operateur_id_donneur
GROUP BY o.nom
ORDER BY portages_sortants DESC;

Quel opérateur perd le plus de clients ?

Refus par motif

SELECT cr.description AS motif, COUNT(*) AS total
FROM porta_portage_data pd
JOIN porta_code_reponse cr ON cr.code = pd.code_reponse
WHERE pd.code_ticket = 200
GROUP BY cr.description
ORDER BY total DESC;

HAVING — Filtrer après le regroupement

WHERE filtre avant le regroupement, HAVING filtre après :

SELECT date_creation, COUNT(*) AS nb_fichiers
FROM porta_fichier
GROUP BY date_creation
HAVING COUNT(*) > 20
ORDER BY nb_fichiers DESC;

Jours où plus de 20 fichiers ont été échangés (jours à fort trafic).

Combiner GROUP BY et JOIN

SELECT
    o.nom AS operateur,
    p.portage_etat AS etat,
    COUNT(*) AS total
FROM porta_portage p
JOIN porta_dossier d ON d.id = p.dossier_id
JOIN porta_operateur o ON o.code = d.operateur_id_receveur
GROUP BY o.nom, p.portage_etat
ORDER BY o.nom, total DESC;

Tableau croisé : pour chaque opérateur receveur, combien de portages dans chaque état.
