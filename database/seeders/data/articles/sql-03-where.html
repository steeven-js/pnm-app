WHERE — Filtrer les résultats

Pourquoi filtrer ?

Sans filtre, SELECT * FROM porta_portage retourne tous les portages. En investigation PNM, on cherche des cas précis : un numéro particulier, un opérateur, une période. C'est le rôle de WHERE.

Syntaxe

SELECT colonnes FROM table WHERE condition;

Comparaisons simples

Égalité

SELECT * FROM porta_operateur WHERE code = 2;

Résultat : uniquement Digicel.

Différent de

SELECT * FROM porta_msisdn WHERE operateur_id_actuel != 2;

Tous les numéros qui ne sont PAS chez Digicel.

Supérieur / Inférieur

SELECT * FROM porta_portage WHERE portage_date_souhaitee > '2026-02-01';

Portages prévus après le 1er février 2026.

Combiner des conditions

AND — Les deux conditions doivent être vraies

SELECT * FROM porta_portage
WHERE portage_etat = 'enCours'
  AND portage_date_souhaitee < '2026-02-10';

Portages « en cours » dont la date de portage est passée — potentiellement bloqués.

OR — Au moins une condition doit être vraie

SELECT * FROM porta_portage_data
WHERE code_ticket = 200
   OR code_ticket = 700;

Tous les tickets qui sont soit des refus (200) soit des erreurs techniques (700).

Combiner AND et OR avec des parenthèses

SELECT * FROM porta_portage_data
WHERE code_ticket = 200
  AND (code_reponse = 1 OR code_reponse = 3);

Refus spécifiquement pour RIO invalide ou RIO expiré.

Opérateurs spéciaux

IN — Parmi une liste de valeurs

SELECT * FROM porta_portage_data
WHERE code_reponse IN (1, 3, 10, 20);

Équivalent à 4 conditions OR, mais plus lisible.

BETWEEN — Dans une plage

SELECT * FROM porta_fichier
WHERE date_creation BETWEEN '2026-02-01' AND '2026-02-15';

LIKE — Recherche partielle (pattern matching)

SELECT * FROM porta_fichier
WHERE nom LIKE 'PNMDATA_02_%';

Tous les fichiers envoyés par Digicel (code 02). Le % remplace n'importe quel texte.

IS NULL / IS NOT NULL

SELECT * FROM porta_portage_data WHERE code_reponse IS NOT NULL;

Tickets qui ont un code de réponse (la colonne n'est pas vide).

Cas pratique PNM

Trouver tous les portages bloqués (en cours depuis plus de 5 jours) :

SELECT id, portage_msisdn, portage_date_souhaitee, portage_etat
FROM porta_portage
WHERE portage_etat = 'enCours'
  AND portage_date_souhaitee < CURRENT_DATE - INTERVAL '5 days'
ORDER BY portage_date_souhaitee;
Exercices :
  1. Trouvez les numéros chez Orange : SELECT * FROM porta_msisdn WHERE operateur_id_actuel = 1
  2. Trouvez les fichiers d'un répertoire donné : SELECT * FROM porta_fichier WHERE repertoire = 'PNMSYNC'
  3. Trouvez les refus RIO invalide : SELECT * FROM porta_portage_data WHERE code_ticket = 200 AND code_reponse = 1
  4. Trouvez les jours fériés de mai 2026 : SELECT * FROM porta_ferryday WHERE ferryday BETWEEN '2026-05-01' AND '2026-05-31'
