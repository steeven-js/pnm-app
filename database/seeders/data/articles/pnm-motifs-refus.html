Motifs de refus d'éligibilité au portage

Lorsque l'OPD reçoit une demande de portage (ticket 1110), il effectue des contrôles d'éligibilité. Si l'un de ces contrôles échoue, le ticket 1210 est émis avec un code motif de refus. Voici les motifs les plus fréquents et les actions correctives associées.

Motif 01 — RIO invalide

Le RIO transmis dans le ticket 1110 ne correspond pas à celui enregistré pour le MSISDN chez l'OPD. Causes possibles : erreur de saisie, RIO expiré après changement de SIM, RIO d'une ancienne ligne.

Action corrective : demander au client de rappeler le 3179 pour obtenir un RIO à jour, puis resoumettre la demande.

Motif 02 — Numéro inactif ou résilié

Le numéro n'est plus actif sur le réseau de l'OPD. Le client a peut-être déjà résilié sa ligne, ou la ligne a été suspendue pour impayé puis résiliée automatiquement.

Action corrective : vérifier avec le client le statut de son ancienne ligne. Si la résiliation est récente, le numéro peut encore être récupérable sous certaines conditions auprès de l'ancien opérateur.

Motif 03 — Portage déjà en cours

Une demande de portage est déjà active pour ce numéro. Il ne peut y avoir qu'un seul portage en cours pour un même MSISDN à un instant donné.

Action corrective : attendre la clôture du portage en cours (bascule ou annulation), puis resoumettre. Vérifier dans PortaDB si un ticket ouvert existe pour ce numéro.

Motif 04 — Numéro non attribué

Le MSISDN n'existe dans aucune base de l'OPD. Le numéro n'a jamais été attribué ou il y a une erreur dans le numéro transmis.

Action corrective : vérifier le numéro exact avec le client. Contrôler le format (10 chiffres commençant par 06 ou le format international 590/594/596).

Motif 05 — Opposition ou restriction judiciaire

Le numéro fait l'objet d'une mesure judiciaire interdisant tout changement d'opérateur (réquisition, mise sous scellé, etc.).

Action corrective : ce motif ne peut pas être contourné. Le client doit résoudre la situation juridique avant de pouvoir porter son numéro.

Motif 06 — Date de portage invalide

La date JP proposée ne respecte pas le délai minimum de 2 jours ouvrés après JD, ou tombe un jour non ouvré (week-end, jour férié local).

Action corrective : resoumettre avec une date JP valide. Vérifier le calendrier des jours ouvrés de la zone Antilles-Guyane (attention aux jours fériés locaux spécifiques).

Motif 07 — Données incohérentes

Les informations du ticket 1110 présentent des incohérences : le code opérateur du RIO ne correspond pas à l'OPD ciblé, le format du MSISDN est incorrect, ou des champs obligatoires sont manquants.

Action corrective : vérifier la cohérence entre le RIO (code opérateur), le MSISDN et l'OPD destinataire. Corriger les données et resoumettre.

Bonnes pratiques

Pour minimiser les refus, il est recommandé de :

  • Toujours demander un RIO fraîchement obtenu (moins de 7 jours)
  • Vérifier le format du RIO avant soumission (12 caractères alphanumériques)
  • Contrôler la clé de vérification avant envoi du ticket 1110
  • S'assurer que le numéro est bien actif en demandant au client s'il peut encore passer des appels
